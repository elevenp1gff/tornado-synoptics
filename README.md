# 龙卷风 · Tornado Deck

单文件 HTML 课堂演示——龙卷风的天气学解读，21 页横向翻页 PPT。浏览器打开即用，无需安装任何软件。

## 技术栈

Three.js 粒子系统 · CSS PPI 雷达 · SVG 环境图 · CSS 多阶段生命周期动画

## 使用

```bash
# 任意浏览器打开
open tornado-deck.html

# 翻页：键盘左右箭头 / 鼠标滚轮 / 触屏滑动
```
